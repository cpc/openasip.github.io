47/**
48 * This class represents cost estimates for a machine configuration.
49 *
50 * Class stores estimates of area (in gates), longest path delay (in nano
51 * seconds) and energy consumptions (in milli joules)
52 * when the configuration is used to run different programs. Area and longest
53 * path delay are constants to one configuration while there can be multiple
54 * programs executed so multiple energy consumption estimations and cycle
55 * counts as well. Each energy consumption is bound to one program.
56 */

66 double area() const;

72private:

74 std::map<const TTAProgram::Program*, double> energyMap_;
75 /// Area estimation value (in gates).
76 double area_;
