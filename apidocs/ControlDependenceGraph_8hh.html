class  ControlDependenceGraph
 

Detailed Description

Declaration of prototype control dependence graph of TTA program representation.

Known limitations are:

  • can not create CDG for procedure with indirect jump - multiple out edges to basic blocks in CFG without edge predicates
  • can not create CDG for procedure with unreachable basic block - crt0 with missing fall through edge after calling __exit
